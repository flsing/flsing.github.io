Circle the Dot game

Built in Java

Eyob Tilaye and I implemented our own version of the popular Circle the Dot game. We implemented the model-view-controller design pattern. We designed the application utilizing event-driven programming. We used a Breadth-First Search algorithm in order to find the shortest path. Simalar examples of the game can be found here.

You can download the code at Github.com.
  • Date: January 2016 - April 2016
  • Client: ITI 1121
  • Category: Model-View-Controllor Game
